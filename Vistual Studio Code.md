## Shortcuts

* Ctrl + ù = Opens the Console
* Ctrl + µ = Clones selected tab
* Ctrl + K, Ctrl + Z = Zen Mode
* Ctrl + K, Ctrl + T = Select Theme
* Alt + Shift + F = Tidy
* Alt + Shift + Left/Right = Selected nested
* Alt + Shift + Up/Down = Duplicates line
