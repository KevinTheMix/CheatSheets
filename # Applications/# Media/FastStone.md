# FastStone

## Quick Tips

* Use layer lock to prevent edition & **selection**

## Glossary

* **FastStone Capture** = take screenshot, pick color (`Ctrl + Shift + P`)
* **FastStone Photo Resizer** = batch convert/rename image files
* **FastStone Image Viewer** = view & edit images

## FastStone Image Viewer

* `D` = draw (add shapes or highlight)
* `H` = flip horizontally
* `I` = show EXIF properties
* `L` = rotate left
* `O` = clone & repair
* `R` = rotate right
* `S` = start diaporama
* `V` = flip vertically
* `W` = open image in file explorer
* `X` = resize (cut)
* `F2` = rename
* `F3` = open batch convert/rename (à la FastStone Photo Resizer)
* `F4` = open batch convert/rename (à la FastStone Photo Resizer)
* `F5` = refresh
* `F6` = send email
* `F7` = randomize image order
* `F10` = reach menu bar
* `F11` = full screen
* `F12` = parameters
* `Alt + G` = quick screen fit rescale
* `Alt + I` = show file properties
* `Ctrl + K` = edit canvas size
* `Ctrl + R` = rescale
* `Ctrl + W` = set as wallpaper
* `Ctrl + Alt + H` = flip horizontally & save (lossless)
* `Ctrl + Alt + L` = rotate left & save (lossless)
* `Ctrl + Alt + R` = rotate right & save (lossless)
* `Ctrl + Alt + V` = flip vertically & save (lossless)
* `Ctrl + Drag` = select area (first change normal mode in _Edit > Selection_)
* `Ctrl + Shift + O` = blur selection
* `Shift + G` = quick 50% rescale
