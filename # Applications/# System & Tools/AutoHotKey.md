# AutoHotKey

Windows inputs (keyboard & mouse) macros/scripting.

## API

* Key selector = `ScrollLock::` (no curly braces) or `^j::`
* Special characters
  * `{#}`
  * `{Left}`, `{Right}`, `{Up}`, `{Down}`
  * `{Delete}`
  * `{BackSpace}`
  * `{Space}`
  * `{Tab}`
* Modifiers = apply to the following key/input
  * Alt = `!`
  * Ctrl = `^`
  * Shift = `+`
  * Ctrl + Shift = `^+`
  * Windows = `#`
