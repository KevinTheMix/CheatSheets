# AutoHotKey

Windows inputs (keyboard & mouse) macros/scripting.

## API

* `{#}`
* `{Left}`, `{Right}`, `{Up}`, `{Down}`
* `{Delete}`
* `{BackSpace}`
* `{Space}`
* `{Tab}`
* Modifiers = aply to the following key/input
  * Alt = `!`
  * Ctrl = `^`
  * Shift = `+`
  * Ctrl + Shift = `^+`
  * Windows = `#`
