* [American McGee's Alice](https://www.myabandonware.com/game/american-mcgee-s-alice-452)
  * [How to install HD version](https://steamcommunity.com/sharedfiles/filedetails/?id=645862676)
* [Command & Conquer](https://cncnz.com/features/freeware-classic-command-conquer-games) = freeware C&C, Red Alert (several releases; some require setting up 16-bit Windows XP compatibilty)
* [Dwarf Fortress](http://www.bay12games.com/dwarves) = ASCII roguelike
* [Item Abuse](https://www.smwcentral.net/?p=section&a=details&id=11342) = Super Mario World Kaizo
* [Octodad](https://younghorses.itch.io/octodad) = QWOP-level ragdoll controls
* [OpenRA](https://www.openra.net) = C&C, Red Alert, Dune 2000
* [Solarus](https://www.solarus-games.org) = FOSS game engine (written in C++) to create Zelda-like games
* [Spongy](https://www.pouet.net/prod.php?which=53871) = 128b Windows XP 3D fractal demoscene
* **StepMania** = free arrow/platform dance/rhythm game (DDR clone)
* [Tasty Static](https://www.tastystatic.com) = Polygonal WipeOut-like
* [Teeworlds](https://teeworlds.com) = Multiplayer 2D platform-shooter
* [Warsow](https://warsow.net) = Quake III -like
