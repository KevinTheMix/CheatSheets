  * Select song > Export (only selected will get exported)
  * Wav > Stems => Will explode the track into stems for mixing.

## Shortcuts

* Alt + t = Name section on Timeline (Chorus, Verse)
* Alt + Left = Reverse
* Ctrl + R = Reverb
* F4 = New Pattern
* Ctrl + C = Copy
* Ctrl + V = Paste
* Ctrl + B = Copy/Paste
