## Shortcuts

* Alt + t = Name section on Timeline (Chorus, Verse)
* Alt + Left = Reverse
* Ctrl + R = Reverb
* F4 = New Pattern
* Ctrl + C = Copy
* Ctrl + V = Paste
* Ctrl + B = Copy/Paste
