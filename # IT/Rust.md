## [Glossary](https://doc.rust-lang.org/cargo/appendix/glossary.html)

* **Cargo** = Rust package manager
* **Cargo Target** = artifacts that will be produced by Cargo packages
* **Crate** = either a library (_library crate_) or an executable program (_binary crate_), can contain modules
* **Module** = logical unit providing isolated namespace within code
* **Package** = collection of source files and a _Cargo.toml_ manifest file describing that package, with a name & version
