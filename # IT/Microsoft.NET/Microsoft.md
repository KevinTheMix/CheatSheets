# Microsoft

## Quick Tips

* [Microsoft Learn](https://learn.microsoft.com/en-us/users/me) = gamified documentation by topics
* [Microsoft Learn: Microsoft Credentials](https://learn.microsoft.com/en-us/credentials)= certifications & applied skills
* [Microsoft Research Tools](https://www.microsoft.com/en-us/research/tools) = index of code/datasets/models/tools created by Microsoft, shared with the broader academic community

## Glossary

* **Channel 9** = (defunct) Microsoft website for employees videos/podcasts
* **Microsoft Ignite** = annual tech & networking conference
* **Microsoft Power BI** = Microsoft's analytics & visualization reporting
