## Glossary

* _.ipa_ = iOS application archive file packaging an iOS app for installation or distribution
* **Apple Developer Program**
* **TestFlight** = platform for distributing/testing beta versions of apps before public release (100 internal/10000 external testers, 90 days, $99/year)
* **Xcode** = suite of developer tools (including an IDE of same name, compiler/debugger, emulators, signing) for building apps on Apple devices (macOS, iOS, iPadOS, watchOS, tvOS, visionOS)
