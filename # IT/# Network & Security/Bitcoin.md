# Bitcoin

First decentralized cryptocurrency.

## Quick Tips

* [Learn Me A Bitcoin](https://learnmeabitcoin.com)

## Glossary

* **Satoshi** = 1/100 000 000 BTC
* **Segregated Witness** (SegWit) = soft fork change in Bitcoin transaction format (prevent nonintentional transaction malleability & increase limits eg block size)
* **Stablecoin** = not necessarily stable cryptocurrency type that aims to maintain a stable value relative to a specific asset or pool/basket of assets (eg a fiat currency, commodity, other cryptocurrencies)
* **UTXO** (Unspent Transaction Output) = past input transaction that is available for outspending
* Properties
  * **Non-Custodial** = self-service (you control your private keys)
  * **Permissionless** = you don't require permission to use the system (eg create an address, send/receive funds, runs software to verify rules yourself)

* _Mt. Gox_ = a defunct Japanese exchange, once the largest worldwide (handling over 70% of transactions), from 2010 until 2014 when it filed for bankruptcy after having lost hundreds of thousands of bitcoin
