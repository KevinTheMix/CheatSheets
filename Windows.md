# Windows

Copy/Move/Shortcut

    Drag&Drop + Shift/Ctrl/Alt

Group renaming

    Multi-select Files > Rename

## Shortcuts

Remote Desktop

    Windows + R > mstsc

Screenshot (Win 10)

  Windows + Shift + S

Switch Virtual Desktop (Win 10)

    Windows + Ctrl + Left/Right

Open new application Window

    Middle-click taskbar App icon
